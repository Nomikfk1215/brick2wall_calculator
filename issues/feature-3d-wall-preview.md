# Feature: 增加 3D 自由视角排砖预览

## 背景

当前页面已经能根据墙长、墙高、墙厚输出灰缝尺寸、砖块数量和 2D 排布图，但客户同学反馈：2D 图很难看出墙体厚度和错缝效果。

## 期望

为页面增加一个 3D 自由视角预览区：

- 每块标准砖渲染为长方体。
- 半砖使用不同颜色。
- 墙长、墙高、墙厚方向都能体现。
- 支持鼠标拖拽旋转、滚轮缩放。
- 页面指标和 3D 图使用同一份排布数据。

## 约束

- 保持最小依赖。
- 优先复用 `src/calculator.js` 输出的 `bricks` 数据。
- 不要重写灰缝计算和砖块数量计算。
- 可以选择原生 CSS 3D，也可以在 Plan 中说明为什么引入 Three.js CDN。

## 验收

- 页面出现 3D 预览区。
- 拖拽和缩放可用。
- 整砖和半砖颜色区分清晰。
- 修改后说明如何验证功能。
